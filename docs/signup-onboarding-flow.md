# Signup & Onboarding Flow — Agent Command Deck

## Flow Overview
```
Landing Page → Create Account → Set Up Workspace → First Workflow Wizard → See Value → Invite Team → Dashboard
```

## Step 1: Create Account
- Email + password (min 8 chars, 1 uppercase, 1 number)
- Google OAuth option
- Verification email with magic link

## Step 2: Workspace Creation
- Company name
- Team size (Just me / 2–5 / 5–20 / 20+)
- Industry dropdown (Tech, Fintech, Healthcare, Staffing, Other)

## Step 3: First Workflow Wizard
Three options:
- **A:** Connect ATS (Greenhouse, Lever, Ashby)
- **B:** Upload a test CV (AI scores against sample job req)  
- **C:** Forward a CV from email (auto-generate inbox address)

Default for new users: Option B with a sample job req "Senior Software Engineer"

## Step 4: See Value Immediately
- AI scores the CV in <10 seconds
- Shows match % (87%), skills matched, skills missing, experience level
- "Go to Dashboard" CTA

## Step 5: Invite Team
- Email invite form (up to 5 for Growth trial)
- Shareable workspace link

## Post-Onboarding: First-Run State
- Kanban board with 1 candidate (from test)
- Onboarding checklist (4 of 5 complete)
- Tooltip tour: 4-step walkthrough
- Suggested next action: "Set up job requirements"
